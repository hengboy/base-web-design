# Public UI I18n Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a lightweight Chinese/English i18n layer for shared layout UI and a top-right language toggle that persists the user's selection.

**Architecture:** Introduce a local React i18n provider with two in-repo dictionaries and wire only shared layout components to translation keys. Keep component registry titles untouched so the change stays narrow and easy to verify.

**Tech Stack:** React 19, TypeScript, Vite, localStorage, existing layout components

---
