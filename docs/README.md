# Docs 索引

`docs/` 目录用于存放本项目的补充说明、组件使用指引和面向开发者的约定文档。

## 当前文档

### [AI 组件选择指南](./ai-component-guide.md)

适用场景：

- 需要根据业务场景快速判断应该使用哪个组件
- 需要让 AI 或开发者按统一标准选择组件
- 需要从“操作、输入、反馈、导航、展示、浮层、菜单、布局”等维度查找现有能力

文档内容包括：

- 组件选择决策树
- 按分类整理的组件索引
- 常见交互和组合模式说明

## 使用建议

建议按下面的顺序查阅文档：

1. 先看项目根目录的 `README.md`，了解框架定位、页面入口和复用方式
2. 再看 `ai-component-guide.md`，根据实际场景选择合适组件
