## 当前实现约定

为了避免文档与代码脱节，这里明确几个现状：

- 组件演示页统一位于 `src/pages/components/`
- 路由、标题、分组、图标由 `src/pages/components/registry.ts` 统一驱动
- 左侧菜单使用分组展示，不再使用旧的混合演示页结构
- `Dashboard` 是首页启动台，不是完整组件总览页
- `Charts` 作为独立页面保留

## 推荐阅读顺序

1. 先看根目录 `README.md`，确认项目结构、首页定位和路由组织
2. 再看 `ai-component-guide.md`，为具体页面或功能选择合适组件
